## 1. 说明

- 本文档基于当前代码和 2026-04-24 的联调冒烟结果整理。
- 前端请求代码里使用的是去掉前缀后的路径，实际后端地址统一为 `/api/v1` + 下表中的接口路径。
- 本次浏览器串行冒烟使用账号：`admin / 123456`。
- 参数化路由本次验证样例：`snapshotId=3`。
- 全站公共接口：
  - `POST /api/v1/auth/login`
  - `GET /api/v1/auth/me`
  - `GET /api/v1/system/dictionaries/cache`（依赖字典缓存的页面会间接使用）
